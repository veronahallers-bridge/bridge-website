# Bridged — ai-bridge.restaurant

**Connecting restaurant managers with the infinite possibilities of AI.**

Bridged shows restaurant managers exactly how to use AI to reduce their daily workload and increase profitability. From handling everyday tasks in seconds to setting up autonomous AI agents that can work for you around the clock.

## Structure

```
/index.html          → Redirects to /en/
/en/index.html       → English site
/de/index.html       → German site (coming soon)
/robots.txt          → Search engine directives
/sitemap.xml         → XML sitemap with hreflang
/_headers            → Cloudflare Pages security headers
/_redirects          → Cloudflare Pages redirects
```

## Hosting

Hosted on Cloudflare Pages. Deploys automatically on push to `main`.
